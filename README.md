# Space Tourism Insurance

## Overview

Space Tourism Insurance is a comprehensive parametric insurance platform designed specifically for commercial space flights. This innovative blockchain-based system provides automated coverage for launch delays, technical failures, and passenger safety concerns, leveraging smart contracts to deliver transparent and efficient insurance solutions.

## 🚀 Project Description

The space tourism industry represents the next frontier in human travel and exploration. However, with this exciting opportunity comes unprecedented risks and uncertainties. Our parametric insurance system addresses these challenges by providing:

- **Real-time Risk Assessment**: Automated monitoring of weather conditions, space traffic, and launch windows
- **Comprehensive Safety Monitoring**: Continuous spacecraft systems monitoring and pre-flight safety checks
- **Instant Compensation**: Automated refunds and compensation for delayed or cancelled flights
- **Transparent Operations**: All insurance policies and claims are handled through immutable smart contracts

## 🎯 Core Features

### Parametric Insurance Model
Unlike traditional insurance that requires lengthy claim investigations, our parametric model automatically triggers payouts based on predefined, objective parameters such as:
- Weather conditions at launch site
- Space traffic congestion
- Technical system failures
- Launch window availability

### Smart Contract Architecture
The system consists of three primary smart contracts:

1. **Launch Conditions Oracle**: Monitors real-time environmental and operational conditions
2. **Vehicle Health Monitor**: Tracks spacecraft systems and safety metrics
3. **Flight Cancellation Compensation**: Manages automated payouts and refunds

## 🚦 Getting Started

### Prerequisites
- Clarinet CLI installed
- Node.js and npm
- Git

### Installation
```bash
# Clone the repository
git clone [repository-url]

# Navigate to project directory
cd Space-Tourism-Insurance

# Install dependencies
npm install

# Run contract checks
clarinet check
```

### Testing
```bash
# Run all tests
clarinet test

# Run specific contract tests
clarinet test --filter launch-conditions-oracle
```
